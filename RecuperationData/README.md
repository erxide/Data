# Recuperation Data

le but du projet est de trouver un plage de donnée et la traité, nous avons pensé alors aux données liés a une competition sportive, comme le [Tournoi des 6 Nations](https://fr.wikipedia.org/wiki/Tournoi_des_Six_Nations). Il existe plusieurs sites web qui partages les statisques du tournoi, alors nous nous sommes dirigé vers le [site officiel](https://www.sixnationsrugby.com/en/m6n/stats/2024?tab=teams) de cette compétition.
